# 卸载图像渲染{#uninstalling-image-rendering}

如果需要，您可以卸载图像渲染。

卸载可能会删除安装文件夹中的某些用户文档。 在卸载之前备份要保留的所有文件。

在卸载之前，请使用管理或根权限登录到服务器主机。
