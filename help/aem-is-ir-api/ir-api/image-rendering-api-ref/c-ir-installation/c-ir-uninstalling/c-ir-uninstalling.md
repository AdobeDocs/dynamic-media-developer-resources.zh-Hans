# 正在卸载图像渲染{#uninstalling-image-rendering}

如果必要，可以卸载图像渲染。

卸载可能会删除安装文件夹中的某些用户文档。 请在卸载之前备份所有要保留的文件。

在卸载之前，请使用管理员或root权限登录到服务器主机。
