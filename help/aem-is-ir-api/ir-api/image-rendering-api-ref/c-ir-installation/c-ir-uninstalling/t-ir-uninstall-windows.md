# 在Windows上卸载{#uninstalling-on-windows}

按照以下说明卸载Windows系统上的图像渲染。

1. 通过运行` *[!DNL install_folder]*\_uninst\uninstaller.exe`启动卸载程序，或从“开始菜单”中选择&#x200B;**[!UICONTROL 项目]** > **[!UICONTROL Scene7]** > **[!UICONTROL 图像渲染]** > **[!UICONTROL 卸载]**。
