# Licensing{#licensing}

需要Scene7图像渲染许可证密钥，并且必须在安装图像渲染的每个系统上安装该许可证密钥。

没有有效的许可证，图像渲染将无法运行。 安装过程包括获取和安装许可证密钥的步骤。
