# 许可{#licensing}

需要Dynamic Media图像渲染许可证密钥，并且必须在安装了图像渲染的每个系统上安装。

如果没有有效的许可证，图像渲染将不会运行。 安装过程包括获取和安装许可证密钥的步骤。
