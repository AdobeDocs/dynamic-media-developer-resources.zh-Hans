# 许可{#licensing}

需要Dynamic Media图像渲染许可证密钥，并且必须在安装图像渲染的每个系统上安装该密钥。

如果没有有效的许可证，则无法运行图像渲染。 安装过程包括获取和安装许可证密钥的步骤。
