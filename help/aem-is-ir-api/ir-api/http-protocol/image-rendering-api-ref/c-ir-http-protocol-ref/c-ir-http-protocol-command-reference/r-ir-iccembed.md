# iccEmbed{#iccembed}

嵌入ICC配置文件。 指定是否应在回复图像中嵌入正在使用的ICC颜色配置文件或使用`icc=`指定的配置文件。

`iccEmbed=0|1`

## 属性 {#section-45c08073626a4cc7823d423dc35b4479}

它可能会出现在请求中的任何位置。

## 默认 {#section-68ed1f0c68dc46b58972804799c277b7}

`iccEmbed=0`，用于在输出图像中不嵌入ICC配置文件。
