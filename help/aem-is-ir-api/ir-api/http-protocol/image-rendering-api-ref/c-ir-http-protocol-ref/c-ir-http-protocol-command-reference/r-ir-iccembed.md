# iccEmbed{#iccembed}

嵌入ICC用户档案。 指定是应将工作ICC颜色用户档案还是使用icc=指定的用户档案嵌入到回复图像中。

`iccEmbed=0|1`

## 属性 {#section-45c08073626a4cc7823d423dc35b4479}

可能在请求中的任何位置发生。

## 默认 {#section-68ed1f0c68dc46b58972804799c277b7}

`iccEmbed=0`, for no embedding of ICC用户档案 in output images.
