除非在详细说明中另有说明，否则服务器返回MIME类型为&lt;text/plain>的文本响应。

`debug`

执行指定的命令并返回渲染的图像。 如果发生错误，则返回错误和调试信息，而不是错误图像(`attribute::ErrorImagePath`)。

`contents`

返回晕影中对象层次的XML表示形式，包括所选对象属性。 请求中的其他命令将被忽略。

`img`

执行指定的命令并返回渲染的图像。 回复数据格式和响应类型由`fmt=`确定。

`imageprops`

返回在URL路径中指定的晕影文件或目录条目的选定属性。 有关回复语法和响应MIME类型的说明，请参阅[属性](../../../../../ir-api/http-protocol/image-rendering-api-ref/c-ir-http-protocol-ref/c-ir-http-protocol-response-data/c-ir-properties.md#concept-e99f1a373eae4af9b41842ca0088ad3a)。 请求中的其他命令将被忽略。 返回以下属性：

`map`

返回晕影中包含的图像映射数据。 默认情况下，返回所有最外面的组的映射数据。 可以使用

`req=map&groupLevel=-1`

映射数据不会缩放到`wid=`或`hei=`，或以其他方式修改。 响应MIME类型为`<text/xml>`。

响应数据由`<map>`元素组成，该元素包含一组`<area>`元素，与HTML `<AREA>`标签类似。

每个`<area>`元素都包括标准`type=`和`coord=`属性，以及指定晕影组名称或名称路径的`name=`属性。 如果相应对象组的蒙版具有不连续区域，则存在多个同名`<area>`元素。

除了默认属性之外，如果创作了晕影，还可以定义其他属性。 此类自定义属性定义为对象组属性。 自定义属性的名称必须以`map`开头。 要包含在`<area>`元素中。 例如，如果组属性包含`map.href=http://www.scene7.com`，则相应的`<area>`元素将包含`href="http://www.scene7.com"`。

如果晕影不包含映射文档，则返回一个XML，其中`<map>`元素为空。

`object`

执行指定的命令并返回由剩余对象选择（使用请求中最后一个`sel=`或`obj=`命令选择的组或对象）所遮罩的渲染图像。 通常与支持alpha的图像格式一起使用（请参阅[fmt=](../../../../../ir-api/http-protocol/image-rendering-api-ref/c-ir-http-protocol-ref/c-ir-http-protocol-command-reference/r-ir-fmt.md#reference-4c743f67d56b47c5b774fcc900ff758c)）。 如果使用的图像格式不支持Alpha，则蒙版外部的区域为黑色。

`props`

执行指定的命令并返回晕影属性和组或对象属性，而不是渲染的图像。 有关回复语法和响应MIME类型的说明，请参阅[属性](../../../../../ir-api/http-protocol/image-rendering-api-ref/c-ir-http-protocol-ref/c-ir-http-protocol-response-data/c-ir-properties.md#concept-e99f1a373eae4af9b41842ca0088ad3a)。 除非也指定了`obj=`或`sel=`，否则将应用默认选择（请参阅[ `obj=` ](../../../../../ir-api/http-protocol/image-rendering-api-ref/c-ir-http-protocol-ref/c-ir-http-protocol-command-reference/r-ir-obj.md#reference-31e7dac7931b4e0eb3c7589f120a1e6a)）。

`userdata`

返回`vignette::UserData`的内容。 服务器将用行终结器(`<cr><lf>`)替换`vignette::UserData`中所有出现的`'??'`。 回复格式化为文本数据，响应MIME类型设置为&lt;text/plain>。

如果在URL路径中指定的对象未解析为有效的晕影映射条目，或者`vignette::UserData`为空，则回复将仅包含行终结器(`CR/LF`)。

将忽略请求字符串中的任何其他命令。

## 属性 {#section-e44092e190ff4f6380583e8ed6ea5b0b}

请求命令。 可能出现在请求字符串中的任意位置。

## 默认 {#section-00c593cbf1af4364b6b78812e6b93c64}

如果URL不包括图像路径或修饰符，则：

```
#S7Z OK 
#Mon Aug 18 17:28:32 PDT 2014 
copyright=Copyright (c) 1995-2014 Adobe Systems Incorporated. All rights reserved.
```

否则，`req=img`
