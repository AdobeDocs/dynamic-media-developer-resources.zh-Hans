## 属性 {#section-2345b2508273426295ce8ac46182ea64}

材质属性。 如果晕影不包含3D反射数据或在晕影中禁用光泽效果，则忽略此项。

## 默认 {#section-0989055fb74a41a3b2f2a47fe7d90a42}

`catalog::Type`如果材料基于目录条目。 否则`type=0`。 如果未指定或`type=0`，则服务器会根据目标对象和其他材质属性选择合适的默认值。
