## 属性 {#section-2345b2508273426295ce8ac46182ea64}

材料属性。 如果暗角不包括3D反射数据或在暗角中禁用了光泽效果，则忽略。

## 默认 {#section-0989055fb74a41a3b2f2a47fe7d90a42}

`catalog::Type` 如果物料基于目录条目。否则`type=0`。 如果未指定，或者如果`type=0`，则服务器将根据目标对象和其他材料属性选择合适的默认值。
