# illum{#illum}

光照映射选择器。 指定此材料更喜欢用来渲染的照明映射。

`illum=-1|0|1|2`

如果指定的照明映射在目标晕影中不可用，则使用最近的可用映射。

`illum=-1` 指定根据值自动选择照明 `gloss=` 图。

## 属性 {#section-aace8466566e4cf1a0c5a6c0167245c9}

材料属性。 如果晕影未定义多个照明映射，则忽略。

## 默认 {#section-c96ecfb232074e80b6a29076f5199403}

`illum=-1`
