# 锐化{#sharpen}

锐化回复图像。 在渲染的图像根据wid=或hei=进行缩放后，可以应用可选锐化。

`sharpen=0|1`

 <tr class="strow"> 
  <td class="stentry"> <p>0 </p></td>

 </tr> 
 <tr class="strow"> 
  <td class="stentry"> <p>1 </p></td> 
  <td class="stentry"> <p>锐化。 </p></td> 
 </tr> 
</table>
