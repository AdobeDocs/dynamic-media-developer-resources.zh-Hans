随机镶边模式(14...18)可用于从不容易重复的图像合成纹理；该算法将根据原始图像创建完全随机或部分随机的纹理。

## 属性 {#section-262bf540930d4890b678ea00cefe1909}

材料属性。 被纯色、倾斜和机柜材料忽略。

## 默认 {#section-e5bbd7d9fbb74852849e605d20f550bb}

`catalog::Repeat`，如果材料基于目录条目，则为 `0` “直线重复”。
