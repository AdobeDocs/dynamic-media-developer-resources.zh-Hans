随机面组模式(14...18)可用于从不容易重复的图像合成纹理；该算法基于原始图像创建完全随机或部分随机纹理。

## 属性 {#section-262bf540930d4890b678ea00cefe1909}

材质属性。 已被纯色、贴花和机箱材料忽略。

## 默认 {#section-e5bbd7d9fbb74852849e605d20f550bb}

`catalog::Repeat`，如果材料基于目录条目，否则`0`（直接重复）。
