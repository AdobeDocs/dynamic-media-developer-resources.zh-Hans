# 光泽效果{#gloss-effects}

如果在晕影中启用此选项，则渲染器会更改光照算法以模拟材料光泽度的变化（主要是通过控制对比度）。

这种效果与反射渲染无关，并通过模拟镜面反射来补充。 效果由相同的控制 `gloss=`， `glossmap=`、和 `type=` 用于控制反射的属性。
