# 光泽效果{#gloss-effects}

如果在暗角中启用，则渲染器将改变照明算法以模拟材料光泽度的变化（主要通过控制对比度）。

此效果与反射渲染无关，通过模拟镜面反射来补充它。 该效果由用于控制反射的相同`gloss=`、`glossmap=`和`type=`属性控制。
