# 光泽效果{#gloss-effects}

如果在晕影中启用该功能，则渲染器改变照明算法以模拟材料光泽度的变化（主要是通过控制对比度）。

这种效果与反射渲染无关，并通过模拟镜面反射来补充这种效果。 效果由用于控制反射的相同`gloss=`、`glossmap=`和`type=`属性控制。
