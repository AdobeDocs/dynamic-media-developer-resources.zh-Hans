# 光泽效果{#gloss-effects}

如果在晕影中启用了这种方法，则渲染器会改变照明算法以模拟材料光泽度的变化（主要通过控制对比度）。

此效果与反射呈现无关，并通过模拟镜面反射来补充它。 用相同的方法控制效果 `gloss=`, `glossmap=`和 `type=` 用于控制反射的属性。
