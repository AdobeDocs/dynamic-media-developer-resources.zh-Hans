# 光泽效果{#gloss-effects}

如果在晕影中启用此功能，则渲染器将改变照明算法以模拟材料光泽度的变化（主要通过控制对比度）。

此效果与反射呈现无关，并通过模拟镜面反射来补充它。 使用用于控制反射的相同`gloss=`、`glossmap=`和`type=`属性来控制效果。
