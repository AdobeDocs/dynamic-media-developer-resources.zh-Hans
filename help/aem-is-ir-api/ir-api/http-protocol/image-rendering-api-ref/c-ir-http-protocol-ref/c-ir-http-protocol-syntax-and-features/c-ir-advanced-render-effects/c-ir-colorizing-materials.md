# 着色材料{#colorizing-materials}

大多数材料都可以动态着色。

着色算法简单易行，对色调范围有限的材料图像效果最好。 要对材料着色，渲染器只需减去`bgc=`值，并将`color=`值添加到每个像素值。

如果未指定`color=`，则禁用着色。 `bgc=` 被橱柜材料忽视；而是使用嵌入在文件 [!DNL vnc] 中的基色值。
