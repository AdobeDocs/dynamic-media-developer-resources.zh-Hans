# 限制{#restrictions}

嵌套和嵌入有一些限制。

为获得良好的服务器性能，嵌套请求返回的图像分辨率应与应用材料的对象的纹理分辨率合理匹配。

外部图像在本地缓存。 只有本地缓存条目失效（基于过期的HTTP头）后，才会检测对此类图像所做的任何更改。
