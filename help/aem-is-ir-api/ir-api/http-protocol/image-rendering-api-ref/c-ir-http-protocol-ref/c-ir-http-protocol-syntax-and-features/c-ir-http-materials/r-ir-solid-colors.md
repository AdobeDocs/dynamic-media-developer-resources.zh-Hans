# 纯色{#solid-colors}

该材料类别包括用于壁、木制品、装饰品、汽车和不需要纹理的任何其它应用的油漆。

例如，可以使用纯色来改变服装项目上不需要改变纹理的颜色，或者为汽车内饰的塑料装饰和地毯设置颜色。 如果MSS中不存在`src=`值，则材料被视为纯色。

纯色可应用于所有可渲染对象。
