# 可重复的纹理{#repeatable-textures}

可重复的纹理包括内部和外部材料，如织物（包括服装和内饰）、墙到墙地板覆盖物、墙纸、台面材料、木纹纹理、屋面和壁板材料，以及任何其他通用纹理。

可重复的纹理可应用于平面、流线、草图、平面、壁和机柜对象。 当应用于非纹理对象时，将使用 `color=` (或 `bgc=` if `color=` 未指定)。

如果材料包含 `src=` 属性，指定图像，以及图像是否出现在除贴图或壁边框以外的MSS中。

在渲染时，通过匹配 `anchor=` 纹理材料的点与对象的纹理原点（如晕影中创作的）。
