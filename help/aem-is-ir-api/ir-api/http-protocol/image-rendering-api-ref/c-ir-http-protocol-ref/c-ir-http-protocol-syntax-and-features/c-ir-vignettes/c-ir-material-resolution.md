# 材质分辨率{#material-resolution}

可重复纹理或其他材质的分辨率值定义场景坐标空间中材质图像的大小。

纹理分辨率值以像素/英寸（或晕影为其场景坐标空间指定的任何单位）指定。

材料分辨率指定为 `res=`.
