# 物质分辨率{#material-resolution}

可重复纹理或其他材料的分辨率值定义场景坐标空间中材料图像的大小。

纹理分辨率值以每英寸像素（或晕影为其场景坐标空间指示的任何单位）指定。

物质分辨率由 `res=`.
