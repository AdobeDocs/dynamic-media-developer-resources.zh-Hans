# 材料分辨率{#material-resolution}

可重复纹理或其他素材的分辨率值定义场景坐标空间中素材图像的大小。

以像素/英寸（或暗角为其场景坐标空间指定的单位）为单位指定纹理分辨率值。

材料分辨率用`res=`指定。
