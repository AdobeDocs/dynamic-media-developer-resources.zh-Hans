# 材料分辨率{#material-resolution}

可重复纹理或其他材料的分辨率值定义场景坐标空间中材料图像的大小。

纹理分辨率值以每英寸的像素数（或暗角为其场景坐标空间指定的单位）来指定。

材料分辨率用`res=`指定。
