# 名称{#name}

配置文件快捷方式名称。 颜色配置文件的快捷方式名称。

## 属性 {#section-52a4b91bbfa045c49c9541444fad2027}

文本字符串。 必需. ICC配置文件映射的主索引键。 每个`icc::Name`值在表中必须唯一，且不得包含“，”字符。

## 默认 {#section-9d23d9b082f541b5b29b4ff92e942942}
