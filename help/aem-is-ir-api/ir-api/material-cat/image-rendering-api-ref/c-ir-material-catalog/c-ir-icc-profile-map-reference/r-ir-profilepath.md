# ProfilePath{#profilepath}

用户档案文件路径。 ICC颜色用户档案文件的路径和名称，包括文件后缀。

## 属性 {#section-5b75d459083f49f3bbd89ce8458559f7}

文本字符串。 必需. 必须是有效的图像服务器文件路径，绝对路径或相对于`attribute::RootPath`的路径。

## 默认 {#section-67b79d62b3e24c62aff6844cc5e04251}

无。
