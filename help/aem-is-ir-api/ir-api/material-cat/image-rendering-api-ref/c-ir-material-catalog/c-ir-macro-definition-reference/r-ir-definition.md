# 定义{#definition}

宏定义。 由一个或多个完整命令组成的渲染服务协议片段，以“&amp;”字符分隔。

宏定义不能引用其他宏。 宏可以定义和使用自定义变量。

## 属性 {#section-2584a79bd2a04ddeaa2c04fdac4835ae}

文本字符串。 必需.

## 默认 {#section-4ab127aa653b4f76afa867c1acb1f31c}

无。
