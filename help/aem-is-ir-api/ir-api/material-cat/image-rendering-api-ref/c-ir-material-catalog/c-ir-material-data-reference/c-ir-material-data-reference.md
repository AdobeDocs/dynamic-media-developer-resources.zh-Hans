# 材料数据引用{#material-data-reference}

如果物料目录需要物料数据，则必须在属性：:CatalogFile中指定物料数据文件的相对或绝对路径。

特定材料目录的材料数据文件中的条目覆盖默认目录的材料数据文件中的条目。
