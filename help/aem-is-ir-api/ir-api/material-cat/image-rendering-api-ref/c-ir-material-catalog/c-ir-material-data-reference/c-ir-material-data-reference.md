# 材质数据引用{#material-data-reference}

如果材料目录需要材料数据，则必须在attribute：：CatalogFile中指定材料数据文件的相对路径或绝对路径。

特定材料目录的材料数据文件中的条目会改写缺省目录的材料数据文件中的条目。
