# 重复{#repeat}

纹理重复模式。 指定纹理图像如何平铺以填充目标表面。

## 属性 {#section-cef4109cddf54ce095c3293d85bc412d}

枚举。 仅用于可重复纹理。 已忽略所有其他材料。
