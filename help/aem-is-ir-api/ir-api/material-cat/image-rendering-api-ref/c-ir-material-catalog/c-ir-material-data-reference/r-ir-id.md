# Id{#id}

目录记录标识符。 渲染服务器用于查找材料记录的键值。 通常是简短且唯一的材料标识符，如SKU编号。

## 属性 {#section-9b20bf2545314620808b4210bce7afc9}

文本字符串。 必需。 每个`catalog::Id`值在此材质目录中必须是唯一的，且不得包含“，”字符。

## 默认 {#section-6a517046c43a40a7ba0af0212b5f31d2}

无。
