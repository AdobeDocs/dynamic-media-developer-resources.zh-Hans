# 伊卢姆{#illum}

照明图选择器。 允许在呈现此材料时使用照明图的显式选择。

## 属性 {#section-162bcf562ca844ccba9e81e267508cca}

枚举。 将设置为–1，以便根据catalog::Gloss的值自动选择照明图。

设置为0、1或2以选择照明图A、B或C。渲染器将选择晕影中可用的最近照明图。

## 默认 {#section-ac386d31ef90423b8a367010a60bddc7}

-1（自动选择）
