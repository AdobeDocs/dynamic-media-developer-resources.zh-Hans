# 基色{#basecolor}

主图像颜色。 减色颜色，用于纹理上色。 应为纹理图像中的平均主色。

如果将没有`catalog::Color`的纹理材质应用于不可纹理的晕影对象，则还可以使用`BaseColor`作为填充颜色。

## 属性 {#section-d9aec235f28b402581cf2277f5fc6fbb}

颜色。 已被纯色和机箱材料忽略。

## 默认 {#section-ca1813a6fcb24645b5835af0891b279d}

`0x808080` （中性灰色）。
