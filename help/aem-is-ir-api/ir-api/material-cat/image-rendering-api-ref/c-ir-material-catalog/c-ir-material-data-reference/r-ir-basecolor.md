# BaseColor{#basecolor}

主要图像颜色。 减色，用于纹理着色。 应是纹理图像中的平均主色。

如果将没有`catalog::Color`的纹理材料应用于非可纹理晕影对象，则也可以使用`BaseColor`作为填充颜色。

## 属性 {#section-d9aec235f28b402581cf2277f5fc6fbb}

颜色. 被实色和机柜材料忽略。

## 默认 {#section-ca1813a6fcb24645b5835af0891b279d}

`0x808080` （中灰色）。
