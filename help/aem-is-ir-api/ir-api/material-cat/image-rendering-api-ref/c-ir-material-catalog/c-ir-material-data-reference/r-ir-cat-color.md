# 颜色{#color}

材质颜色。 固体色材料的主色和彩色质构材料的添加色。

## 属性 {#section-073772b43b264b37bd96d1fb7404d10b}

颜色。 纯色材料所必需的。 对于可重复纹理、橱柜和窗口覆盖材料而言，为指定上色颜色而选择此选项。 如果不修改可重复纹理或机箱材料的颜色，此字段必须留空。

## 默认 {#section-bf2c2ec642ed47da880ae2943b7d2c18}

无。
