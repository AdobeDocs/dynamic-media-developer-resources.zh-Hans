# 分辨率{#resolution}

默认分辨率。 提供默认分辨率，以防特定目录记录不包含有效的目录：:Resolution值。

## 属性 {#section-06d519158b9f479896f945747c670736}

实数，大于0。 通常以每英寸像素数表示，但也可以以其他单位表示，如每米像素数。

## 默认 {#section-eea922c37c224e1dbcab3bc53ee13aca}

如果未定义或为空，则从`default::Resolution`继承。
