# IccDither{#iccdither}

颜色转换抖动。 指定在未使用进行显式选择时，是否应使用抖动来提高颜色转换的感知质量 `icc=`.

## 属性 {#section-646fb48084734c66bf648360f3a5bfd1}

标记. 设置为 `0` 禁用或 `1` 启用抖动。

## 默认 {#section-c9066c361215404d847f4d2c8f1ea3a5}

继承自 `default::IccDither` 如果未定义或为空。
