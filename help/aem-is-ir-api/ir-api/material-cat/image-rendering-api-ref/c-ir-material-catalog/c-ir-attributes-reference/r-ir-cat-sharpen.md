# 锐化{#sharpen}

默认回复图像锐化。 指定在编码前缩放回复图像时是否锐化这些图像。

## 属性 {#section-faf57183efa0456fb37a7803a9486248}

标记.

## 默认 {#section-e6f1f81ef5c949a488b2002581bda0d0}

继承自 `default::Sharpen` 如果未定义或为空。
