# VignetteMapFile{#vignettemapfile}

晕影映射数据文件路径。 指定包含此图像目录的晕影映射的文件。

## 属性 {#section-8f624cb8e49b44258a961fb7e464ce0a}

文本字符串值。 相对于目录文件夹的绝对文件路径或路径。

## 默认 {#section-cc5c2e0ba54a4e5890a7330ee038fac8}

空，表示此材料目录不包含晕影映射。
