# AllowDirectURLs{#allowdirecturls}

允许绝对源URL。

设置为 `1` 允许绝对HTTP URL(以“ `http://`“” `src=` 中。 设置为 `0` 禁止绝对URL。

## 属性 {#section-9246bc06c7484f9e8f65a5edb24f6854}

标记.

## 默认 {#section-a881c30ea28e47e296faa890c3bd6daf}

继承自 `default::AllowDirectUrls` 如果未定义或为空。
