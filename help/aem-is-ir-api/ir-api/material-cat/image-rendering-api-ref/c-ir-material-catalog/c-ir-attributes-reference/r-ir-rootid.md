# RootId{#rootid}

目录标识符。 用于在HTTP请求中的暗角、材料或ICC用户档案对象说明符中标识此目录的HTTP路径元素。

## 属性 {#section-c33266223d5b47029df756caa4e0df0c}

文本字符串值。 可能包括在http路径中有效的任何字符。

## 默认 {#section-e0ed902557684345850b86672b5dbe5b}

无. 每个目录必须具有唯一 `catalog::RootId` 值。 default.ini通常为空 `catalog::RootId`。
