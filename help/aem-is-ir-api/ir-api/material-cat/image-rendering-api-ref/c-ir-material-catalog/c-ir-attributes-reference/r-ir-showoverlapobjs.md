# ShowOverlapObjs{#showoverlapobjs}

显示重叠对象。 指定应最初显示还是隐藏重叠对象。

## 属性 {#section-3e82e9b98d7a46f884d8b9798cec35c2}

标记.

## 默认 {#section-e6f7e33a629b4592979bab6d14700e8f}

如果未定义或为空，则从`default::ShowOverlapObjs`继承。
