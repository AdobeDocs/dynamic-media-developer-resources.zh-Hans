# 晕影图引用{#vignette-map-reference}

如果物料目录需要晕影图，则必须在属性：:VignetteMapFile中指定晕影图文件的相对或绝对路径。

特定材料目录的晕影映射文件中的条目将覆盖默认目录的晕影映射文件中的条目。
