# 暗角图参考{#vignette-map-reference}

如果材料目录需要暗角映射，则必须在属性：:VignetteMapFile中指定暗角映射文件的相对或绝对路径。

特定材料目录的晕影映射文件中的条目将覆盖默认目录的晕影映射文件中的条目。
