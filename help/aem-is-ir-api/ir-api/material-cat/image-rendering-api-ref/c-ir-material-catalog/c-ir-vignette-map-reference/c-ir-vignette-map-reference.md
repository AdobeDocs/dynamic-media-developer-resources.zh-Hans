# 晕影映射引用{#vignette-map-reference}

如果材质目录需要晕影映射，则必须在attribute：：VignetteMapFile中指定晕影映射文件的相对路径或绝对路径。

特定材质目录的晕影映射文件中的条目将覆盖默认目录的晕影映射文件中的条目。
