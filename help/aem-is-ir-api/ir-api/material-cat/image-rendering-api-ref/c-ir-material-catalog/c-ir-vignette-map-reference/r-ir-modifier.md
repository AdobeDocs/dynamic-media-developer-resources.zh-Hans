# 修饰符{#modifier}

请求修饰符字符串。 没有或多个以“&amp;”字符分隔的图像渲染命令。 用于存储每次请求此晕影映射条目时要应用的模板和修饰符。

引用此记录的请求或模板中的相同命令会覆盖此字段中的命令。

## 属性 {#section-b7b01c307ebc41d48c34db77307dbfd1}

文本字符串。 可选.

## 默认 {#section-31dd0ce4c0604fbaa7e7f1304626223b}

无。
