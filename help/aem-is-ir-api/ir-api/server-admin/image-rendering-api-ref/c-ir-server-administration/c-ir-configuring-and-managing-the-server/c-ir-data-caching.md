# 数据缓存{#data-caching}

图像渲染利用图像服务缓存功能来保存响应图像数据、用于嵌套/嵌入请求的中间数据以及从外部源获得的源图像数据（例如纹理）。

有关缓存机制的描述，请参阅图像服务文档。
