# 目录文件夹(ir.catalogRootPath){#catalog-folder-ir-catalogrootpath}

材料目录文件夹的位置（必须包含所有catalog.ini文件）。

可以是绝对路径或相对于[！DNL的路径 *[!DNL install_folder]*]。 默认为 [!DNL ./catalog]，对应于[！DNL *[!DNL install_folder]*/catalog]。
