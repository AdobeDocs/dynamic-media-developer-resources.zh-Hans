# 目录文件夹(ir.catalogRootPath){#catalog-folder-ir-catalogrootpath}

材料目录文件夹的位置（所有catalog.ini文件必须位于其中）。

可以是绝对路径，也可以是相对于[!DNL *[!DNL install_folder]*]的路径。 默认值为[!DNL ./catalog]，它对应于[!DNL *[!DNL install_folder]*/catalog]。
