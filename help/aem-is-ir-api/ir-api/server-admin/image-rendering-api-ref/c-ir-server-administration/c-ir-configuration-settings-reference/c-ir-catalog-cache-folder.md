# 目录缓存文件夹(ir.catalogCacheRoot){#catalog-cache-folder-ir-catalogcacheroot}

目录系统缓存的根文件夹。

可以是绝对路径，也可以是相对于[!DNL *[!DNL install_folder]*]的路径。 默认值为[!DNL ./cache]，它对应于[!DNL *[!DNL install_folder]*/cache]。
