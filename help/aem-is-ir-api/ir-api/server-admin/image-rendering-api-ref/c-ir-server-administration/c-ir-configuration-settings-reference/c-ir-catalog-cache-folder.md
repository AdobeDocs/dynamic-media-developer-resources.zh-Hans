# 目录缓存文件夹(ir.catalogCacheRoot){#catalog-cache-folder-ir-catalogcacheroot}

目录系统缓存的根文件夹。

可以是绝对路径或相对于[!DNL *[!DNL install_folder]*]的路径。 默认值为[!DNL ./cache]，与[!DNL *[!DNL install_folder]*/cache]对应。
