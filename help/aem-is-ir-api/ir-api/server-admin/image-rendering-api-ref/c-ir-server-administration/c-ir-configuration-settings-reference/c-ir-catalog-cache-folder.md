# 目录缓存文件夹(ir.catalogCacheRoot){#catalog-cache-folder-ir-catalogcacheroot}

可以是绝对路径或相对于[!DNL *[!DNL install_folder]*]的路径。 默认值 [!DNL ./cache]是，它对应于[!DNL *[!DNL install_folder]*/缓存]。
