# 属性{#properties}

返回属性数据以响应以下`req=`类型：

* `catalogprops`
* `imageprops`
* `props`
* `userdata`

`userdata` 仅当响应的内容采用属性格式时，响应 `catalog::UserData` 才会以属性格式设置。

* [文本(Java)属性](r-text-java-properties.md)
* [JavaScript属性](r-javascript-properties.md)
* [XML属性](r-xml-properties.md)
* [JSONP属性](r-json-properties.md)
