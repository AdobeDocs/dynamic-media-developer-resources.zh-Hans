# img{#img}

图像（默认）。 请求标准图像数据。

`req=img`

回复数据格式和响应MIME类型由`fmt=`确定。 修饰符`req=img`是默认的请求类型，不需要显式使用。 可使用基于`catalog::Expiration`的TTL来缓存HTTP响应。

其他请求命令将按文档所述应用。
