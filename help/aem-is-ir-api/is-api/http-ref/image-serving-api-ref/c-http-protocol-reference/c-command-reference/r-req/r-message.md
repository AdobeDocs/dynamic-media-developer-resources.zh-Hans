# 消息{#message}

客户端消息。 为客户端提供一种机制，以便将短文本消息插入到服务器日志中。

`req=message&message= *`字符串`*`

 <tr class="strow"> 
  <td class="stentry"> <p><span class="varname">字符串</span> </p> </td> 
  <td class="stentry"> <p>消息字符串。 </p></td> 
 </tr> 
</table>

无法缓存HTTP响应。 返回的MIME类型为`text/plain`的响应为空。
