# loadcache{#loadcache}

预加载服务器缓存。 与req=img类似，服务器会执行请求，但不会返回图像，而是返回回复图像的长度(image.length)，格式为MIME类型为text/plain的文本数据。

`req=loadcache`

无法缓存HTTP响应。

请求中的其他命令也适用于所记录的内容。
