# loadcache{#loadcache}

预载服务器缓存。 执行请求的方式与req=img类似，但服务器返回的是返回图像的长度(image.length)，格式为MIME类型为text/plain的文本数据。

`req=loadcache`

HTTP响应不可缓存。

请求中的其他命令将按文档进行应用。
