# 解决{#resolve}

调试请求。 此debug命令解析并预处理请求、执行图像目录查找、目录：：修饰符包含、宏和变量替换等，就像req=img一样。

`req=resolve`

将返回MIME类型为`text/plain`的最终请求字符串，而不是结果图像。

HTTP响应无法缓存。
