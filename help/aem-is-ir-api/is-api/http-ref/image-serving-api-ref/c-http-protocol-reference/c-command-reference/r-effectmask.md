# 效果蒙版{#effectmask}

覆盖请求的attribute：：ApplyEffectMask的值。

effectMask是全局修饰符。 它将应用于请求中的所有效果层。

有效值为0和1。 零(0)表示不应用效果蒙版。 一(1)表示应用效果蒙版。 默认值为1。
