# effectMask{#effectmask}

覆盖请求的属性：:ApplyEffectMask的值。

effectMask是全局修饰符。 它适用于请求中的所有效果层。

有效值为0和1。 0表示不应用效果蒙版。 1表示应用效果蒙版。 默认值为1。
