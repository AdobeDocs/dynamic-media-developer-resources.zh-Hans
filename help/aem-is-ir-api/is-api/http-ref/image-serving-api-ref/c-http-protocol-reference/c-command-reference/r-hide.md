# 隐藏{#hide}

隐藏图层。 允许隐藏图层，但不从请求字符串中删除它。

`hide=0|1`

## 属性 {#section-09ff67290f064411baeb503a1aac2953}

图层属性。 当为指定时，应用于图层0 `layer=comp`。

## 默认 {#section-ec9f5130be1b435c84ba1bba54e38a48}

`hide=0` 对象。
