# 图层顺序{#layer-order}

图层编号还决定了z顺序。

层0（背景层）是必需的；其他层数不需要连续，并且按层数递增的顺序绘制在背景层的顶部。 具有最高图层编号的图层呈现在顶部，且永远不会被其他图层遮盖。
