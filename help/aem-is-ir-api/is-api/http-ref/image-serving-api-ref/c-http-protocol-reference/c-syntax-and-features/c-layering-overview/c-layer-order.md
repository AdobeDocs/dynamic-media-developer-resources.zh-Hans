# 图层顺序{#layer-order}

层号也决定了z顺序。

要求图层0（背景图层）；其他图层编号不需要连续，并且按图层编号的升序在背景图层上方绘制。 具有最高图层编号的图层将渲染到顶部，且永远不会被其他图层遮挡。
