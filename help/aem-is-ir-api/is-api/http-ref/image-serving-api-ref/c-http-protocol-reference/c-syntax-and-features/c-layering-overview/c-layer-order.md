# 图层顺序{#layer-order}

图层编号也决定z顺序。

第0层（背景层）是必需的；其他图层编号无需连续，将按升序图层编号的顺序绘制在背景图层上。 具有最大图层编号的图层将呈现在顶部，并且不会被其他图层遮挡。
