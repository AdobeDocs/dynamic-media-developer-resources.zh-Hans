# coord{#coord}

像素坐标。 用于以相对于图像或图层矩形的左上角的像素偏移的形式指定图像坐标。 这些参数通常用于图像偏移或裁剪参数。

坐标 `0,0` 指图像或矩形的左上角。 增加值会向右下方移动。
