# 图像服务器日志{#image-server-log}

图像服务器将记录写入此日志文件，以记录警告和错误，并记录启动和关闭事件。

此外，可以配置映像服务器日志来跟踪大量其他详细信息，如映像文件访问，以帮助调试服务器问题。

已在[!DNL ImageServerRegistry.xml]中配置映像服务器日志。
