# monitor.conf{#monitor-conf}

包含监视/警报系统的设置。

此文件是JAVA属性文件。 必须注意遵守适当的公约；否则平台服务器可能无法开始。 请特别注意，在Windows文件路径中，必须使用多次反斜杠“\\”或单正斜杠“/”，而不是反斜杠“\”，因为反斜杠在此类文件中用作转义字符。

对此文件所做的更改在保存文件后不久生效。
