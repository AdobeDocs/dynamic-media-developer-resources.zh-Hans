# 堆空间优先级警报{#heap-space-priority-alert}

当可用Java堆空间在Java垃圾收集周期之后紧接在指定的阈值之下时，将发送优先级警报。

应通过增加Java堆空间来解决重复的警报。 在通过`AS::monitorAlertGenerator.heapSpaceResetInterval`指定的延迟期到期之前，后续出现此条件不会生成电子邮件警报。
