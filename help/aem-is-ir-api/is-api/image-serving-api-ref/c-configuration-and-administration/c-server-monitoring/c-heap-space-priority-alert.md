# 堆空间优先级警报{#heap-space-priority-alert}

当可用Java堆空间在紧随Java垃圾收集周期之后低于指定阈值时，将发送优先级警报。

应通过增加Java堆空间来解决重复的警报。 在用`AS::monitorAlertGenerator.heapSpaceResetInterval`指定的延迟期已过之前，此条件的后续出现不会导致电子邮件警报。
