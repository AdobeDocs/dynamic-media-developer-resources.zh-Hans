# 出血边距{#bleedmargin}

设置出血边距。 设置PDF文件中设置的出血边距。

`bleedMargin=[ *[!DNL top]*[, *[!DNL left]*= *[!DNL top]*[, *[!DNL bottom]*= *[!DNL top]*[, *[!DNL right]*= *[!DNL left]*]]]]`分

默认情况下，`bleedMargin`设置为由`viewWidth`和`viewHeight`定义的文档的完整大小。 如果未指定，*[!DNL left]*、*[!DNL bottom]*&#x200B;和&#x200B;*[!DNL right]*&#x200B;值将默认为&#x200B;*[!DNL top]*&#x200B;值。
