# mediaMargin{#mediamargin}

设置媒体边距。 设置在PDF文件中设置的媒体边距。

` mediaMargin=[ *[!DNL top]*[, *[!DNL left]*= *[!DNL top]*[, *[!DNL bottom]*= *[!DNL top]*[, *[!DNL right]*= *[!DNL left]*]]]]` 点

默认情况下，`mediaMargin`设置为由`viewWidth`和`viewHeight`定义的文档的完整大小。 如果未指定，则&#x200B;*[!DNL left]*、*[!DNL bottom]*&#x200B;和&#x200B;*[!DNL right]*&#x200B;值将默认为&#x200B;*[!DNL top]*&#x200B;值。
