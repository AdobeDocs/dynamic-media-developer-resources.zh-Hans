# ProfilePath{#profilepath}

用户档案文件路径。 ICC颜色用户档案文件的路径和名称，包括文件后缀。

## 属性 {#section-d342866853f24e168ba44215ef594e29}

文本字符串。 必需. 必须为有效的图像服务器文件路径，绝对路径或相对路径 `attribute::RootPath`。

## 默认 {#section-838c08a4443648ffb2f5b9255fafa8dc}

无。
