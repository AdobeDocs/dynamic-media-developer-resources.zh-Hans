# 配置文件路径{#profilepath}

配置文件路径。 ICC颜色配置文件文件的路径和名称，包括文件后缀。

## 属性 {#section-d342866853f24e168ba44215ef594e29}

文本字符串。 必需. 必须是有效的图像服务器文件路径，可以是绝对路径，也可以是相对路径 `attribute::RootPath`.

## 默认 {#section-838c08a4443648ffb2f5b9255fafa8dc}

无。
