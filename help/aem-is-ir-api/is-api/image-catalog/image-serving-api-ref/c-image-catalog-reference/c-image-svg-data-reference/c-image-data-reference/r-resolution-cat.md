# 分辨率{#resolution}

对象分辨率。 全尺寸图像的对象分辨率。

用于使用`res=`命令进行基于分辨率的缩放。

## 属性 {#section-861a86da50364e2181d62778338359ea}

实数，大于0。 通常以每英寸的像素数表示，但也可以以其他单位表示，如每米的像素数。 可选。

## 默认 {#section-5b3a83d6410b43efa1fa3add396196f9}

`attribute::Resolution` 如果字段不存在、值为0或字段为空，则使用。
