# PrintResolution{#printresolution}

打印分辨率。 打印全尺寸图像的分辨率。

此值嵌入在回复图像标题中，除非被`printRes=`覆盖。

## 属性 {#section-de3c1f73da7b43208beeec841c1778c1}

整数，大于0。 以每英寸点数表示。 可选。

## 默认 {#section-0cac992554ec4247ab05f70d9840a045}

`attribute::PrintResolution` 如果字段不存在，则使用该字段（如果值为0或字段为空）。
