# AssetType{#assettype}

资源类型. 用于标识发布到目录的集的类型：:ImageSet。

资产类型决定为`req=set`请求生成的响应类型。 如果未指定值，自动检测规则将确定`req=set`响应类型。

## 属性 {#properties}

以下集的枚举值：

MEDIASET

旋转集

SPINSET2D

视频集

图像集

ECATALOG

## 默认 {#section-b94ce61423194729918456103ef3f72c}

无。
