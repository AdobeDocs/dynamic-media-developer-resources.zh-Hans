# AssetType{#assettype}

资源类型. 用于标识发布到catalog::ImageSet的集类型。

资产类型确定要为`req=set`请求生成的响应类型。 如果未指定值，则自动检测规则将确定`req=set`响应类型。

## 属性 {#properties}

以下集中的枚举值：

MEDIASET

旋转集

SPINSET2D

VIDEOSET

图像集

ECATALOG

## 默认 {#section-b94ce61423194729918456103ef3f72c}

无。
