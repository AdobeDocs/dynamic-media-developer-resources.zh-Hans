# 字体路径{#fontpath}

字体文件路径。 主体字体文件的路径和名称，包括文件后缀。

## 属性 {#section-ae2b7daf74a4419ebcec8f13aa8c1462}

文本字符串。 必需. 必须为有效的图像服务器文件路径，绝对路径或相对于`attribute::RootPath`的路径。

## 默认 {#section-3fb42f362de84fdb8ba9fe4cc9628aa3}

无。
