# PsName{#psname}

Postscript字体的名称。

目前，只有FXG渲染器才能通过字体的Postscript名称引用字体。

## 属性 {#section-a560f8bfc69a4d88aeb0b53c1625153b}

文本字符串。 可选. 字体映射的辅助索引键。 相同 `font::PsName` 值只能在相同的字体映射文件中出现一次。

## 默认 {#section-c5903d5340e740b69ec0150d2303a893}

无。
