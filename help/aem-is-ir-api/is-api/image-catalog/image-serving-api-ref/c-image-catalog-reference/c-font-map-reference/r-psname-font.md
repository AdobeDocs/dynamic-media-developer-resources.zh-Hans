# PsName{#psname}

Postscript字体名称。 Postscript字体名称。

此时，只有FXG渲染器可以通过其Postscript名称引用字体。

## 属性 {#section-a560f8bfc69a4d88aeb0b53c1625153b}

文本字符串。 可选。字体映射的次索引键。 相同的`font::PsName`值只能在同一字体映射文件中出现一次。

## 默认 {#section-c5903d5340e740b69ec0150d2303a893}

无。
