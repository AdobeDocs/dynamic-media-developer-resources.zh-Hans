# 字体映射引用{#font-map-reference}

如果图像目录需要字体映射，则必须在以下属性中指定字体映射文件的相对路径或绝对路径：:FontMapFile。

特定图像目录的字体映射中的条目将覆盖默认目录的字体映射中的条目。

图像提供支持Adobe Type1、OpenType和TrueType字体。

`textPs=` 还支持CID、ATC和位图字体。
