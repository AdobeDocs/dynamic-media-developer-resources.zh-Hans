# 规则集文件{#rule-set-files}

规则集文件是XML格式的文本文件，必须遵循相应的标准和惯例。

[!DNL RuleSet.xsd]安装在默认目录文件夹中，应将其用于在将规则集文件提交到图像服务之前对其进行验证。 应用了严格解析，不符合[!DNL RuleSet.xsd]的规则集文件无法加载。
