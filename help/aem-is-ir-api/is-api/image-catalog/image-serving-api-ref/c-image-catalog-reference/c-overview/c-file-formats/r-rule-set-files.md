# 规则集文件{#rule-set-files}

规则集文件是XML格式的文本文件，必须遵守相应的标准和惯例。

[!DNL RuleSet.xsd] 安装在默认目录文件夹中，在将规则集文件提交到图像服务之前，应该使用来验证这些文件。应用严格解析，且不符合[!DNL RuleSet.xsd]的规则集文件不加载。
