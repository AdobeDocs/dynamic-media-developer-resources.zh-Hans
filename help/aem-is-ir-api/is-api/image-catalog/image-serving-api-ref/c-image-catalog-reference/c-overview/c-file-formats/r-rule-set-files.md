# 规则集文件{#rule-set-files}

规则集文件是XML格式的文本文件，必须符合相应的标准和惯例。

[!DNL RuleSet.xsd] 已安装在默认目录文件夹中，并且应在将规则集文件提交到图像服务之前用于验证它们。应用严格解析，不符合[!DNL RuleSet.xsd]的规则集文件不加载。
