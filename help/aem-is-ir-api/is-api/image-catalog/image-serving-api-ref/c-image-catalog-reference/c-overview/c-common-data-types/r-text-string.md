# 文本字符串{#text-string}

文本字符串数据类型。

字符串分隔符是可选的，除非字符串包含任何`<CR>`、`<LF>`或`<TAB>`字符。 单引号或双引号均可用作分隔符。 如果使用引号，则字符串中嵌入的任何此类引号必须使用两个连续的引号进行转义（例如，本月的特殊引号）。
