# Text string{#text-string}

文本字符串数据类型。

字符串分隔符是可选的，除非字符串包含任何 `<CR>`、 `<LF>`或字符 `<TAB>` 。 单引号或多次引号可用作分隔符。 如果使用引号，则嵌入字符串中的任何此类引号都必须使用两个连续的引号进行转义(例如：本月的特别优惠)。
