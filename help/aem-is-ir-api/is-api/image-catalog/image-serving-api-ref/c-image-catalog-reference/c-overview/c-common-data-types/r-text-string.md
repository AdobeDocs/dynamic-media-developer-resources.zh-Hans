# 文本字符串{#text-string}

文本字符串数据类型。

字符串分隔符是可选的，除非该字符串包含 `<CR>`， `<LF>`，或 `<TAB>` 个字符。 可以使用单引号或双引号作为分隔符。 如果使用引号，则必须使用两个连续的引号对字符串中嵌入的任何此类引号进行转义（例如：本月的特殊引号）。
