# 默认区域设置{#defaultlocale}

默认区域设置ID。 未在请求中指定locale=时用于本地化/翻译目的的区域设置ID。

## 属性 {#section-a101e9658d304e9caf032a99ecb80386}

文本字符串。 如果和 `attribute::LocaleMap` 为空， `attribute::LocaleStrMap` 或者当指定并 `locale=` 非空时忽略。
