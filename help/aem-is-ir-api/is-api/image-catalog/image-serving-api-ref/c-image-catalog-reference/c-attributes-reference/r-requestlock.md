# RequestLock{#requestlock}

请求锁定模式。 指定必须包含在请求中的请求锁的类型。

## 属性 {#section-ae89274d3b7940d0bf3d816f7c2f51fd}

标志。 设置为0可禁用请求锁定，设置为1可选择简单哈希锁定。

## 默认 {#section-fd5b5f0417bd444189b20494d179193e}

如果未定义或为空，则从`default::RequestLock`继承。
