# IccDither{#iccdither}

颜色转换抖动。 指定在未使用icc=进行明确选择时，是否应使用抖动来提高颜色转换的感知质量。

## 属性 {#section-b7ba44d2d5de43f5a0841fe4cac29d35}

标记. 将设置为0禁用抖动，将设置为1启用抖动。

## 默认 {#section-86c4230a16454464880f64d4ab5ad533}

从`default::IccDither`继承（如果未定义或为空）。
