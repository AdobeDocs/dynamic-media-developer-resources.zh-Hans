# IccDither{#iccdither}

颜色转换抖动。 指定在通过icc=未做出明确选择时，是否应使用仿色来提高颜色转换的感知质量。

## 属性 {#section-b7ba44d2d5de43f5a0841fe4cac29d35}

标记. 设置为0可禁用抖动，设置为1可启用抖动。

## 默认 {#section-86c4230a16454464880f64d4ab5ad533}

继承自 `default::IccDither` 如果未定义或为空。
