# 默认字体{#defaultfont}

默认字体名称。 指定text=请求未提供字体时应使用的字体。

## 属性 {#section-b71618fec3c443a392d67eff480f0e5d}

文本字符串。 如果指定，则必须是此图像目录的字体映射或默认目录的字体映射中的有效font：：Name值。

## 默认 {#section-ed0199be32a24e31a47b345ce7d109f6}

如果未定义或为空，则从`default::DefaultFont`继承。
