# TotalStreamBitRate{#totalstreambitrate}

总流比特率。 表示视频资产的总流比特率的长值。

`req=set`和`req=mbrSet`请求用于导出比特率数据。

仅适用于静态内容目录。

## 属性 {#section-76731467b23240ac89b0767491c6020b}

整数值(bps)

## 默认 {#section-dba669ea75354671bbdb3b520e2f6911}

无。
