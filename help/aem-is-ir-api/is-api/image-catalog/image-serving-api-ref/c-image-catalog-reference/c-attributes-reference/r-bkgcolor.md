# BkgColor{#bkgcolor}

## 属性 {#section-2768b2beee214676aa32f17e2a0e3343}

颜色.

## 默认 {#section-9c2c5794b82a4ef89a09551f08d03dbc}

从`default::BkgColor`继承（如果未定义或为空）。
