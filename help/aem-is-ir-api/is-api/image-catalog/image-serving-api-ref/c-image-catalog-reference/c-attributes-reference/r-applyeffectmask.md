# ApplyEffectMask{#applyeffectmask}

（可选）在效果图层上应用最终蒙版，以遮住效果与图层之间的重叠区域。

有效值为0和1。 0表示不应用效果蒙版。 1表示应用效果蒙版。 默认值为1。
