# RuleSetFile{#rulesetfile}

规则集定义文件路径。 指定包含此图像目录的规则集定义的文件。

## 属性 {#section-e8449b2225294518a5f9ddb698f68360}

文本字符串值。 相对于目录文件夹的绝对文件路径或路径。

## 默认 {#section-20a9d535d64347abaf0233595b2ad089}

为空，表示此图像目录没有专用请求预处理规则。
