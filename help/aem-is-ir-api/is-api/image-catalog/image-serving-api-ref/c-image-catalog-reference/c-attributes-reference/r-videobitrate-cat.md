# 视频比特率{#videobitrate}

视频比特率。 允许指定与每个视频记录相关联的比特率。

使用者 `req=set` 和 `req=mbrSet`请求导出比特率数据。

## 属性 {#section-76731467b23240ac89b0767491c6020b}

整数值(bps)

## 默认 {#section-dba669ea75354671bbdb3b520e2f6911}

无。
