<!--<a id="section_2768B2BEEE214676AA32F17E2A0E3343"></a>-->

`Bfc=true|false`

  <td class="stentry"> <p>指定浏览器格式转换功能 — 将JPEG或PNG图像转换为较小的浏览器特定文件格式 — 开启(<span class="codeph"> true </span>)还是关闭(<span class="codeph"> false </span>)。 </p> </td> 
 </tr>
