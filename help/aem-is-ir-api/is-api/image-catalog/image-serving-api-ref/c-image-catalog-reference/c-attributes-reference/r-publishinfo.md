# 发布信息{#publishinfo}

目录注释字符串。 可用于标识目录版本、发布日期等的文本字符串。

该字段的内容返回自 `req=catalogprops`. 此属性不在其他情况下使用。

## 属性 {#section-73ff2a7c2e5b4c7a952453276f5bed23}

文本字符串。

## 默认 {#section-82e5a885e142414f996c609efbb962b1}

继承自 `default::PublishInfo` 如果未定义或为空。
