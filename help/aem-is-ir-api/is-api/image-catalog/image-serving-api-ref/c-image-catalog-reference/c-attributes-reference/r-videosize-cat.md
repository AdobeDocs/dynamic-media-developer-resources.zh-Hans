# VideoSize{#videosize}

视频大小。 允许指定与每个视频记录关联的大小。

`req=set`请求用于导出视频大小数据。

## 属性 {#section-93a2631a57e2423ea6b40f68c55062a1}

宽度和高度的以逗号分隔的整数值（例如，512,288）。

## 默认 {#section-baaee759ef344c54bbc014dc8944768a}

无。
