# 安装图像服务 {#installation}

安装图像服务包括两个主要的服务器组件：

* 图像服务器. 独立的本机代码可执行文件。
* 平台服务器。 独立的Java™ Web应用程序和应用程序服务器。

图像渲染是这两个服务器组件的一部分。

在后续说明中， `install_root` 指安装了“图像提供”和“图像呈现”的根文件夹。 和 `server:port` 是指配置为平台服务器的主侦听端口的主机名或IP地址和端口号。
