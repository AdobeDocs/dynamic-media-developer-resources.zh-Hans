# 安装{#installation}

必须先安装图像服务，然后才能使用。

图像提供包括两个主要组件：图像服务器是独立的本机代码可执行文件，而平台服务器是独立的Java Web应用程序和应用程序服务器。 图像渲染是这些服务器组件的一部分。

在后续说明中，`install_root`是指安装了图像服务和图像渲染的根文件夹，`server:port`是指配置为平台服务器主侦听端口的主机名或IP地址和端口号。
