# 在Linux上启动或停止{#starting-or-stopping-on-linux}

在Linux上启动或停止图像服务有两个选项。

* 开始、停止和验证图像服务状态的脚本位于[!DNL ImageServing/bin]文件夹中：

   `ImageServing.sh {start|stop|restart|status}`
* 系统管理员应熟悉以下替代方法：

   `/sbin/service ImageServing {start|stop|status}`
