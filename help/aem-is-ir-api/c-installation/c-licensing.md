# Licensing{#licensing}

要使用图像服务，必须在要使用图像服务的每个系统上安装Scene7图像服务许可证密钥。

同样，要使用图像渲染，需要Scene7图像渲染许可证密钥。 图像渲染无需图像服务许可证即可使用，反之亦然。

安装过程包括获取和安装必要的许可证密钥的步骤。
