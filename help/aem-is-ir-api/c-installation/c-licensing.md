# 许可{#licensing}

要使用图像服务，需要Dynamic Media图像服务许可证密钥，并且必须在要使用图像服务的每个系统上安装。

同样，要使用图像渲染，需要Dynamic Media图像渲染许可证密钥。 图像渲染可以在没有图像服务许可证的情况下使用，反之亦然。

安装过程包括获取和安装必要的许可证密钥的步骤。
