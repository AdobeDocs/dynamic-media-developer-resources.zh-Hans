# 键盘辅助功能和导航{#keyboard-accessibility-and-navigation}

查看器UI中公开的所有功能均可通过键盘访问。
最终用户可以使用 `Left`, `Right`, `UP` 和 `DOWN` 击键来平移图像。
的 `Esc` 可使用击键重置图像。
