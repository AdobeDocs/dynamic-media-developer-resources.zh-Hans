# 主查看器区域{#main-viewer-area}

主视图区域是缩放图像所占用的区域。 当未指定大小时，通常将其设置为适合可用的设备屏幕。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

**主查看器区域的CSS属性**

使用以下CSS类选择器控制查看区域的外观：

```
.s7interactiveimage
```

示例——设置具有白色背景( `#FFFFFF`)的查看器，并使其大小为1174 x 500像素。

```
.s7interactiveimage { 
 background-color: #FFFFFF; 
 width: 1174px; 
 height: 500px;  
}
```
