# 缩放视图{#zoom-view}

主视图由静态图像组成。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

**主查看器区域的CSS属性**

通过以下CSS类选择器控制查看区域的外观：

```
.s7interactiveimage .s7zoomview
```

示例 — 使主视图透明。

```
.s7interactiveimage .s7zoomview { 
 background-color: transparent; 
}
```
