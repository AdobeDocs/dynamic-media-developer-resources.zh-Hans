# 传送视图{#carousel-view}

主视图由横幅图像组成。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

**主查看器区域的CSS属性**

使用以下CSS类选择器控制查看区域的外观：

```
.s7carouselviewer .s7carouselview
```
