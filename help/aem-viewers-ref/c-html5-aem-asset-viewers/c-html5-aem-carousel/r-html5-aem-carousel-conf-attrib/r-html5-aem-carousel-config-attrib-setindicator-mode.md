# SetIndicator.mode{#setindicator-mode}

`[SetIndicator.|<containerId>_setIndicator.]mode=numeric|dotted`

<table id="table_0BEA0B5FFDF64E5594B534B2A87A6D88"> 
 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 数字|点数</span> </p> </td> 
   <td colname="col2"> <p> 配置设置指示符的呈现样式。 </p> <p>设置为虚线 <span class="codeph"> 时</span> ，组件会为所有页面呈现相同的指示符。 </p> <p>设置为数字 <span class="codeph"> 时</span> ，它在每个指示符元素中放置一个基于1的页码。 </p> <p>数字 <span class="codeph"> 操作模式</span> ，在能够进行触摸输入的设备上不受支持。 而是使用此类设备 <span class="codeph"> 上的点</span> 。 </p> </td> 
  </tr> 
 </tbody> 
</table>
