# Dynamic Media 查看器参考指南{#dynamic-media-viewers-reference-guide}

自定义 Dynamic Media 查看器，以在桌面设备和触摸屏设备上使用。您可以使用的媒体类型包括基本缩放、ecatalog、缩放、ecatalog搜索、弹出、内联缩放、混合媒体、旋转、视频、轮播、维度、交互式视频、交互式图像和视频360查看器。
