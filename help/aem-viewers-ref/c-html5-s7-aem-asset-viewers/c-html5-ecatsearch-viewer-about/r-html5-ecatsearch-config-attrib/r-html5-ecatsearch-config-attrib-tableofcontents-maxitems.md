# TableOfContents.maxitems{#tableofcontents-maxitems}

   <td> <p> <span class="codeph"><span class="varname"> maxitems</span></span> </p> </td> 
   <td> <p>下拉目录中的最大项目数。 </p> <p>此外，您还可以减少下拉列表中可见项目的数量，以防其被外部容器裁掉。 </p> <p>当设置为 <span class="codeph"> 0时</span> ，该组件会根据其添加到的容器显示尽可能多的项目。 </p> </td>
