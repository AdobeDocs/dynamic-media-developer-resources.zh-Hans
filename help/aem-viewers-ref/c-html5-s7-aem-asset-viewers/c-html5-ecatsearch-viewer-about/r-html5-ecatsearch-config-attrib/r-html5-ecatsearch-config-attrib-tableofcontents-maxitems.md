  <tr> 
   <td> <p> <span class="codeph"><span class="varname"> maxitems</span></span> </p> </td> 
   <td> <p>目录下拉列表中的最大项目数。 </p> <p>此外，您还可以减少下拉列表中可见项目的数量，以防其被外部容器裁掉。 </p> <p>当设置为<span class="codeph"> 0</span>时，如果组件添加到的容器，该组件会显示尽可能多的项目。 </p> </td>
