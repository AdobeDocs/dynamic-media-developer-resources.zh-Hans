 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0 | 1 </span> </p> </td> 
   <td colname="col2"> <p> 切换后，滚动按钮会自动导致色板跳转整页长度。 </p> <p>如果色板不适合，则最后一页上会显示额外的空白。 此外，最后一页的单元格数量与前一页相同。 </p> <p>将忽略scrollstep，并且鼠标滚动仅在整个页面上完成。 </p> </td>
