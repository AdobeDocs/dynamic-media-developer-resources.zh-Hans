# getComponent{#getcomponent}

Spin Viewer的JavaScript API参考

`getComponent(componentId)`

返回对查看器使用的查看器SDK组件的引用。 网页可以使用此方法扩展或自定义现成查看器的行为。 仅在查看器回调运行之 `initComplete` 后调用此方法，否则查看器逻辑可能尚未创建组件。

使用SDK API时，务必按照查看器SDK中的说明使用正确、完全限定的SDK [命名空间](../../../c-html5-s7-aem-asset-viewers/c-html5-spin-viewer-about/c-html5-spin-viewer-namespace.md#concept-fa293878c9ff4758ae888415c70fbeef)。

有关特定组件的更多信息，请参阅查看器SDK API文档。

## Returns {#section-1d3cf85bc7cc4dfe9670e038d02b9101}

`{Object}` 对查看器SDK组件的引用。 如果查看器 `null` 组件不受 `componentId` 支持，或者查看器逻辑尚未创建该组件，则此方法将返回。

## 示例 {#section-9e9332aa86b74a5fb321375c03fdc5b3}

```
<instance>.setHandlers({ 
 "initComplete":function() { 
  var spinView = <instance>.getComponent("spinView"); 
} 
})
```
