# 主查看器区域{#main-viewer-area}

主视图区域是弹出视图和色板所占用的区域。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

**主查看器区域的CSS属性**

查看区域的外观由以下CSS类选择器控制：

```
.s7flyoutviewer
```

示例 — 设置带有白色背景的弹出查看器( `#FFFFFF`)并将其大小设置为260 x 500像素。

```
.s7flyoutviewer { 
 background-color: #FFFFFF; 
 width: 260px; 
 height: 500px;  
}
```
