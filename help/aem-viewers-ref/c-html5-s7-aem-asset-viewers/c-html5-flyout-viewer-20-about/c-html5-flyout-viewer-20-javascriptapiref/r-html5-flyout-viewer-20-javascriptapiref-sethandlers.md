# setHandlers{#sethandlers}

弹出查看器的JavaScript API参考。

`setHandlers(handlers)`

指定零个或多个回调处理程序。 对此方法的调用将完全覆盖以前为该查看器实例分配的事件处理程序。 必须在`init()`之前调用。
