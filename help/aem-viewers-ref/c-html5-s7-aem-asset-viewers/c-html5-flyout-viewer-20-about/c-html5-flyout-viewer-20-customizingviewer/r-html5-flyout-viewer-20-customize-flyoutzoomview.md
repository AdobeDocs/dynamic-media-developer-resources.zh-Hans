# 弹出缩放视图{#flyout-zoom-view}

主视图由静态图像、弹出视图中显示的缩放图像、静态图像上显示的高亮导航区域以及静态图像顶部显示的提示消息组成。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

如果所查看图像的尺寸与弹出缩放视图的尺寸不匹配，则图像内容将居中在弹出缩放视图的矩形显示区域内。

**主视图的CSS属性**

主视图的外观由以下CSS类选择器控制：

```
.s7flyoutviewer .s7flyoutzoomview
```

   <td colname="col2"> <p> 主视图的背景颜色。 </p> </td>

示例——使主视图透明：

```
.s7flyoutviewer .s7flyoutzoomview { 
 background-color: transparent; 
}
```

```
 left: 612px; 
 top: 0px; 
 width: 600px; 
 height: 400px;  
}
```

**主视图中突出显示的CSS属性**

主视图中高亮的外观由以下CSS类选择器控制：

```
.s7flyoutviewer .s7flyoutzoomview .s7highlight
```

可以使用CSS控制背景、边框、透明度和类似属性。 但是，高亮DOM元素的大小和位置由查看器逻辑管理。 不支持通过CSS覆盖它。

示例——设置具有40%透明度和一个像素红色边框的绿色突出显示：

```
.s7flyoutviewer .s7flyoutzoomview .s7highlight { 
 background-color: green; 
 opacity: 0.4;  
 filter: alpha(opacity = 40); 
 border: 1px solid red; 
}
```

**光标的CSS属性**

当`highlightmode`参数设置为`cursor`时，主视图中的突出显示将替换为固定大小的光标图稿，该图稿由CSS类选择器控制：

```
 .s7flyoutviewer .s7flyoutzoomview 
.s7cursor
```

可以使用CSS控制背景图像和大小。

**提示消息的CSS属性**

提示消息的外观由以下CSS类选择器控制：

```
.s7flyoutviewer .s7flyoutzoomview .s7tip
```

可以通过CSS配置字体样式、大小外观和垂直偏移。 但是，水平对齐由查看器逻辑管理。 不支持使用`left`或`right`属性通过CSS覆盖它。

提示消息可以本地化。 有关详细信息，请参阅[用户界面元素的本地化](../../../c-html5-s7-aem-asset-viewers/c-html5-flyout-viewer-20-about/c-html5-flyout-viewer-20-localization.md#concept-6c8e58c611934e93ae3f211f46e15c27)。

示例——要设置半透明提示消息，其中带有白色的Arial 12px字体，距主视图底部50像素偏移，填充和圆边：

```
.s7flyoutviewer .s7flyoutzoomview .s7tip { 
bottom: 50px; 
color: #ffffff; 
font-family: Arial; 
font-size: 12px; 
padding-bottom: 10px; 
padding-top: 10px; 
padding-left: 12px; 
padding-right: 12px; 
background-color: #000000; 
border-radius: 4px; 
opacity: 0.5; 
filter: alpha(opacity = 50); 
}
```
