# 图标效果{#icon-effect}

缩放指示器覆盖在主视图区上。 当图像处于重置状态时，它会显示，并且它还取决于图标效果参数。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

查看区域的外观由以下CSS类选择器控制：

```
.s7ecatalogviewer .s7pageview .s7iconeffect
```

>[!NOTE]
>
>图标效果支持`media-type`属性选择器，您可以使用它在不同设备上应用不同的图标效果。 具体而言，`media-type='standard'`对应于桌面系统，其中鼠标输入通常被使用，`media-type='multitouch'`对应于具有触摸输入的设备。

示例——为桌面系统和触控设备设置100 x 100像素缩放指示器，具有不同的图稿。

```
.s7ecatalogviewer .s7pageview .s7iconeffect { 
 width: 100px; 
 height: 100px; 
} 
.s7ecatalogviewer .s7pageview .s7iconeffect[media-type='standard'] { 
 background-image:url(images/v2/IconEffect_zoom.png); 
} 
.s7ecatalogviewer .s7pageview .s7iconeffect[media-type='multitouch'] { 
 background-image:url(images/v2/IconEffect_pinch.png); 
}
```
