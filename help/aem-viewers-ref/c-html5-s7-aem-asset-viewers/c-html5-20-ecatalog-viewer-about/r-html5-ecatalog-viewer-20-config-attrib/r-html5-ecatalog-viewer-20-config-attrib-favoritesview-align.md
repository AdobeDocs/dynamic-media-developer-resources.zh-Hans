# 收藏夹视图。对齐{#favoritesview-align}

`[FavoritesView.|<containerId>_favoritesView.]align=left|center|right,top|center|bottom`

<table id="table_2B109D2F91E64B5382B31921C3780FA5"> 
 <tbody> 
  <tr> 
   <td colname="col1"> <p><span class="codeph">左|中|右，上|中|下</span> </p> </td> 
   <td colname="col2"> <p> 指定组件区域中缩略图容器的内部水平对齐或定位。 </p> <p>在FavoritesView中，规定了内部缩略图容器的大小，因而只显示缩略图的总量。 因此，内部容器和外部组件边界之间存在一些边距。 </p> <p>此修饰符指定如何在组件内水平放置内部缩略图容器。 </p> </td> 
  </tr> 
 </tbody> 
</table>
