# 下载{#download}

可以使用控制栏中的“下载”按钮将电子目录下载为PDF文件。

出现以下情况时，“下载”按钮在查看器用户界面中自动可用：

* 客户公司中存在实际的PDF文件。
* 该名称与传递到查看器的电子目录资源的名称匹配，并包括 `.pdf` 扩展。
* PDF文件发布在Dynamic Media Classic中。
