# 下载{#download}

可以使用控制栏中的“下载”按钮，将电子目录下载为PDF文件。

发生以下情况时，查看器用户界面中会自动显示“下载”按钮：

* 客户公司中存在实际的PDF文件。
* 该名称与传递到查看器并包含`.pdf`扩展的e-catalog资产的名称匹配。
* PDF文件在Dynamic Media Classic中发布。
