# 下载{#download}

可以使用控制栏中的“下载”按钮将电子目录下载为PDF文件。

当发生以下情况时，查看器用户界面中会自动显示“下载”按钮：

* 客户的公司中存在实际的PDF文件。
* 该名称与传递给查看器并包含扩展名的电子目录资产的名称相 `.pdf` 匹配。
* PDF文件已在SPS中发布。
