# ZoomView.reset{#zoomview-reset}

`[ZoomView.|<containerId>_zoomView.]reset=0|1`

<table id="table_49FFD1BC53B846F09A6D214BC8C5C3FE"> 
 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0 | 1</span> </p> </td> 
   <td colname="col2"> <p> 当帧（图像）更改时重置视图端口。 如果设置为<span class="varname"> 0</span>，则保留当前视图端口，同时保留新设置图像的长宽比，并尽可能保持最佳的适合。 </p> </td> 
  </tr> 
 </tbody> 
</table>

## 属性 {#section-50bcd15223174bb79ce08b31ea03d682}

可选。

## 默认 {#section-7564169749ff4a4996049ea1148cb2a5}

`1`

## 示例 {#section-96e69b70365f461dae4399e49044ea2f}

`reset=0`
