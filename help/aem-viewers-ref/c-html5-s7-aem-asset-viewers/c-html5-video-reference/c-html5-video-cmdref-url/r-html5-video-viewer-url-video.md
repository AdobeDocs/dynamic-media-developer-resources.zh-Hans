## 属性 {#section-f42369774e2740dcb399626a0e4e930e}

必需. (除非使用`asset`参数。 请参阅[外部视频支持](../../../c-html5-s7-aem-asset-viewers/c-html5-video-reference/r-html5-video-viewer-20-external-video-support.md#concept-22c67fee43274a29b28ee16770b1b1f3)。)

## 默认 {#section-d016470e92a74f98a18c4ab3489410a5}

无。
