## 参数 {#section-0302e9238dbc4704914e938f42c881e6}

| 名称 | 类型 | 说明 |
|---|---|---|
| `*`colorManagement`*` | `xsd:string` | 颜色管理模式选择。 |
| `*`sourceProfileHandle`*` | `xsd:string` | 源用户档案句柄。 |
| `*`outputProfileHandle`*` | `xsd:string` | 输出用户档案句柄。 |
