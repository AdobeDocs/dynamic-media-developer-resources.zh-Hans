# ActiveJob{#activejob}

在服务器上运行的作业。 此外，它还是计划作业的实例。

作业存在于3个状态：

* 计划运行。
* 当前正在运行。
* 已完成运行（并已将信息写入作业日志）。

指定要返回作业类型的作业类型值。 您可以返回以下作业：

* `imageServingPublishJob`
* `imageRenderingPublishJob`
* `videoPublishJob`
* `serverDirectoryPublish`
* `JobUploadDirectoryJob`
* `uploadUrlsJob`
