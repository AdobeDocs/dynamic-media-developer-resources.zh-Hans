# ActiveJob{#activejob}

在服务器上运行的作业。 此外，它是计划作业的实例。

就业有三种状态：

* 已计划运行。
* 当前正在运行。
* 已完成运行（并已将信息写入作业日志）。

要返回作业类型，请指定作业类型值。 您可以返回以下作业：

* `imageServingPublishJob`
* `imageRenderingPublishJob`
* `videoPublishJob`
* `serverDirectoryPublish`
* `JobUploadDirectoryJob`
* `uploadUrlsJob`
