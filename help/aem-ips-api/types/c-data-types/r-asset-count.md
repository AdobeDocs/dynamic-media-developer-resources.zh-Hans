# AssetCount{#assetcount}

公司的资产类型和计数。

资产类型包括 [!DNL ViewerSWf], [!DNL PDF], [!DNL Cabinet]，等等。

| 名称 | 类型 | 说明 |
|---|---|---|
| assetType | `xsd:string` | 与计数关联的资产类型。 |
| 计数 | `sd:int` | 数量 `assetType` 资产。 |
