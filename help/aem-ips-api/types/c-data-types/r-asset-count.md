# AssetCount{#assetcount}

资产类型和公司计数。

资产类型 [!DNL ViewerSWf]包括、 [!DNL PDF]、 [!DNL Cabinet]和其他。

| 名称 | 类型 | 说明 |
|---|---|---|
| ` *`assetType`*` | `xsd:string` | 与计数关联的资产类型。 |
| ` *`计数`*` | `xsd:int` | 公司中 `assetType` 的资产数。 |
