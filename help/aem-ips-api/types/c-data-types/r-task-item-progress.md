## 参数 {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

| 名称 | 类型 | 说明 |
|---|---|---|
| ` *`itemName`*` | `xsd:string` | 正在处理的项目的名称。 |
| ` *`progress`*` | `xsd:double` | 进度完成%。 |
| ` *`progressMessage`*` | `xsd:string` | 处理消息。 |
| ` *`lastProgressUpdate`*` | `xsd:dateTime` | 上次更新的时间。 |
