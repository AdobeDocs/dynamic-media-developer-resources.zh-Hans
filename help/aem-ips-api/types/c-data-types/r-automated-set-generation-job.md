**emailSetting选项**

`emailSetting`参数包括以下选项：

| 选项 | 返回结果 |
|---|---|
| `All` | 指定收件人的所有作业通知（错误、警告、完成）。 |
| `Error` | 指定收件人的作业错误。 |
| `ErrorAndWarning` | 指定收件人的作业错误和警告。 |
| `JobCompletion` | 向指定的收件人发出作业完成通知。 |
| `None` | 作业不会向指定的收件人发送任何作业通知。 |
