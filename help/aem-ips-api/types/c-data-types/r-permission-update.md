# PermissionUpdate{#permissionupdate}

描述权限更改。

语法

## 参数 {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

| 名称 | 类型 | 说明 |
|---|---|---|
| ` *`groupHandle`*` | `xsd:string` | 组句柄。 |
| ` *`permissionType`*` | `xsd:string` | 权限类型。 |
| ` *`isAllowed`*` | `xsd:boolean` | 确定是否允许权限更新。 |
| ` *`isOverride`*` | `xsd:boolean` | 确定权限是否覆盖其他权限。 |
