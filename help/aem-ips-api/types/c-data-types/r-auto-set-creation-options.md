语法

## 参数 {#section-0302e9238dbc4704914e938f42c881e6}

|---|---|---|
| autoSetsArray | `types:HandleArray` | [!DNL PropertySet]句柄的数组，用于定义上载期间应用的自动生成集脚本。 |
