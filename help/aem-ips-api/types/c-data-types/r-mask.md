# 蒙版{#mask}

遮罩图像的一部分。 蒙版始终与图像相关联。 从ImageInfo获取蒙版。

语法

## 参数 {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

| 名称 | 类型 | 说明 |
|---|---|---|
| ` *`maskHandle`*` | `xsd:string` | 遮罩手柄。 |
| ` *`名称`*` | `xsd:string` | 蒙版名称。 |
| ` *`maskPath`*` | `xsd:string` | 遮罩的相对路径。 |
| ` *`maskFile`*` | `xsd:string` | 蒙版文件. |
| ` *`lastModified`*` | `types:dateTime` | 上次修改遮罩的日期、时间和时区。 |
