## 参数 {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

| 名称 | 类型 | 说明 |
|---|---|---|
| `*`originalPath`*` | `xsd:string` | 文件路径。 |
| `*`originalFile`*` | `xsd:string` | 文件名. |

| `*`optimizedFile`*` | `xsd:string` | 优化的文件名。 |
