# UsmsharpMaskOptions{#unsharpmaskoptions}

有助于提高优化金字塔TIF文件的图像锐度的设置。

`unsharpMaskOptions=[ *`数量、半径、阈值、单色`*]`

## 参数 {#section-c3f0d03136ba4422819cb463bd393885}

为`minOccurs=" *`n`*".`的`unsharpMaskOptions`选项指定值
