# [!DNL AssetContextStateUpdate]{#assetcontextstateupdate}

为与资产关联的发布上下文设置一组新的发布状态标记。

**参数**

| 名称 | 类型 | 说明 |
|---|---|---|
| assetHandle | `xsd:string` | 处理要更新的资产。 |
| contextStateUpdateArray | `types:ContextStateUpdateArray` | 要更新的资源的发布联系状态数组。 |
