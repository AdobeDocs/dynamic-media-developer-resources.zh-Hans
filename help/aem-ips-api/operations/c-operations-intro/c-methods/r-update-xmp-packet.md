# updateXMPPacket{#updatexmppacket}

设置或更新资源的XMP元数据包。

语法

## 授权用户类型 {#section-ee88a759f4774482a4734201a971f610}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `ImagePortalAdmin`
* `ImagePortalUser`
* `ImagePortalContrib`
* `ImagePortalcontribUser`

## 参数 {#section-7a89621d441840faba639746b410a489}

**输入(updateXMPPacketParam)**

| 名称 | 类型 | 必需 | 说明 |
|---|---|---|---|
| companyHandle | `xsd:string` | 是 | 公司句柄。 |
| assetHandle | `xsd:string` | 是 | 资产句柄。 |
| 压缩包 | `xsd:Base 64 binary` | 是 | 要设置或更新的[!DNL zlib-compressed]个XMP数据包。 |

**输出(updateXMPPacketReturn)**

| 名称 | 类型 | 必需 | 说明 |
|---|---|---|---|
| 成功 | `xsd:boolean` | 是 | 如果数据包已更新，则返回`true`。 |
