# deleteAssetMetadata{#deleteassetmetadata}

删除资产的元数据值。 可与元数据删除数组结合使用，以在批量中设置值。

语法

## 授权用户类型 {#section-e913be43b684491daf02bc73211e4290}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `ImagePortalAdmin`
* `ImagePortalContrib`
* `ImagePortalContribUser`

>[!NOTE]
>
>用户必须具有资产的读取和删除权限。
