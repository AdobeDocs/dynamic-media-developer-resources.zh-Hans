# getXMPPacket{#getxmppacket}

检索指定资产的XMP元数据包。

语法

## 授权用户类型{#section-7cb9c26045214f01b1d6b6948b6c6a18}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `ImagePortalAdmin`
* `ImagePortalUser`
* `ImagePortalContrib`
* `ImagePortalContribUser`

## 参数 {#section-b4075df0e4414b00b961d978d5471db9}

**输入(getXMPPacketParam**

| 名称 | 类型 | 必需 | 说明 |
|---|---|---|---|
| ` *`companyHandle`*` | `xsd:string` | 是 | 要返回的包的公司句柄（例如`c|656`）。 |
| ` *`assetHandle`*` | `xsd:string` | 是 | 应检索XMP数据包的资产。 |

**输出(getXMPPacketReturn)**

| 名称 | 类型 | 必需 | 说明 |
|---|---|---|---|
| ` *`compressedPacket`*` | `xsd:Base 64 binary` | 是 | [!DNL zlib-compressed] XMP数据包。 |
