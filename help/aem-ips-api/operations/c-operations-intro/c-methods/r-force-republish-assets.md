# forceRepublishAssets{#forcerepublishassets}

重置一个或多个资产的发布状态，以强制在下一个发布作业中重新发布资产。

语法

## 授权用户类型 {#section-3d5a3e3afea748d69845de5c8c376448}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `ImagePortalAdmin`
* `ImagePortalContrib`
* `ImagePortalContribUser`
