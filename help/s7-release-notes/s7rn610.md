## Scene7 IPS（图像制作系统）4.5.0 {#section-3e37ed37b6d146419f4578bd7ffeaa74}

**新增功能**

* 现在正在运行JDK 1.8。
* 支持EXIF方向标记。
* 对于临时FTP文件，上载目录文件跳过（使用ProFTP创建）。
* 图像渲染现在支持即时发布。 这意味着您渲染的晕影资产可立即在您的网站上启动，而无需启动发布作业。 如果您有兴趣使用此功能，请联系技术支持，让他们为您的帐户启用即时发布功能。

**不再支持的功能**

* IPS Classic发布验证中的“目录服务器重新同步”选项已删除。
* 不能再设置空或重复的元数据标记字段值。

**错误修复**

* 响应API `getAssets`,`<metadataArray>`文件的[!DNL .vtt]为空。

* 方向缺少userdata字段。
* `getJobLogDetails`对更新ACO文件的作业的响应未正确指定作业详细信息。
* PSD主文件已记录为每个图像服务器多次发布。
* 名称中包含无效字符的资产可以上传到IPS。
* 无法生成的视频缩略图保留为“队列中的待处理”状态。
* 在大型`deleteFolder`和服务器目录上载之间进行文件夹还原的争用条件。
* 在上传时提前发布时，PDF文件在每个发布服务器上被记录两次。
* 上传具有相同层名称的多个PSD文件会导致优化错误。
* 客户PDF的PDF光栅化导致生成的图像出现意外白线。
* 视频缩略图文件在上传时发布多次。
* 重复的上传文件检测会生成两个上传通知，从而引发已完成的文件记帐。
* 视频转码的Sidecar XML文件在更新时多次记录为已发布。
* 当您删除嵌套文件夹并将另一个文件夹（不在嵌套文件夹的父文件夹中，并且与嵌套文件夹同名）移到以前嵌套文件夹的位置时，moveFolder会失败。
* 共享和本地配置中的群集状态配置项未通过清理清除。
* 更改API日志解析器以筛选电子邮件地址格式。
* 视频处理未正确映射`Iptc4xmpCore:Location`。
* 清理和上传之间存在争用条件。
* 发布不使用`assetFile`，因此暂存发布仅适用于新资产。
* 更新规则集时引发空指针。

## 查看器 {#section-833ab92c91c941d2bfdc27f233f582ad}

**Dynamic Media Image Serving Viewers 5.2.2的新增功能、增强功能和错误修复**

查看器升级可向后兼容 — 无需对现有Web代码进行任何更改。 但是，建议您在我们的暂存环境中测试新查看器。 以下网站提供了有关如何设置系统以访问我们的暂存服务器的说明：

[http://helpx.adobe.com/experience-manager/scene7/kb/base/upgrade-management/testing-scene7-viewers-upgrade.html](http://helpx.adobe.com/experience-manager/scene7/kb/base/upgrade-management/testing-scene7-viewers-upgrade.html)

在将计算机设置为访问暂存服务器后，您可以检查网站以测试升级。 如果您使用的是现成的HTML5查看器，最佳实践是针对我们的标准暂存服务器和s7is5-preview-staging.scene7.com进行测试。

新增功能和增强功能

另请参阅[查看器参考指南](https://experienceleague.adobe.com/docs/dynamic-media-developer-resources/library/home.html)。

* 在eCatalog查看器中添加了对“打印”、“下载”和“收藏”的支持。
* 添加了使用getComponent API检索ParameterManager的功能。
* 转换后的旋转查看器、缩放查看器、视频查看器和弹出查看器可用于图稿。
* 添加了对IE11本机全屏的支持。
* 在容器中重构了模拟（非本机）全屏支持。
* 增加了CSS小标记大小以支持更大的手机。
* 在桌面浏览器上，删除了HTML5旋转、基本缩放、缩放、旋转和混合媒体查看器的CSS大小标记。
* 添加了对允许在SpinView中配置预加载帧的质量的支持。
* 已修复 — Galaxy S4纵向模式：启用全屏模式时使用的CSS大小标记不正确。
* 已修复 — 在Internet Explorer 9上将javascript模板用于外部链接后，[eCatalog]图像映射卡住。
* 已修复 — Internet Explorer 9和Internet Explorer 10:如果为CSS中的IMG定义了width属性，则缩放操作会不正确。

常规

* 修复了s7sdk.browser.device.version错误地检测到第五个Android的问题。
* 删除了在SDK组件中使用可见性：可见样式的情况。

FlyoutzoomView

* 已修复 — [IE9]弹出窗口有时在鼠标移开后仍保留在屏幕上。
* 已修复 — 在初始图像中添加了两次iscommand字符串。

## 不再支持的功能 {#section-9295ea9313c84695b19bca0e0f01a690}

* **Flash查看器生命周期终** *止通知自2017年1月31日起，Adobe Scene7(现称Dynamic Media Classic)正式终止对Flash查看器平台的支持。*

* 提醒：自2014年1月31日起，Scene7正式停止对DHTML查看器平台的支持。

   有关此更改的更多信息，请参阅以下常见问题解答网站：

   [http://dev.day.com/content/docs/en/cq/current/administering/integrating_with_omniture/scene7/dhtml-viewer-endoflifefaqs.html](http://dev.day.com/content/docs/en/cq/current/administering/integrating_with_omniture/scene7/dhtml-viewer-endoflifefaqs.html)

* Scene7Creative Suite扩展在此版本中被弃用，因为Creative Cloud应用程序不支持Flash面板。 内容摄取工作流的进一步开发将重点关注Adobe Experience Manager Dynamic Media和Creative Cloud。 对于Scene7,Scene7发布系统中的上传和导出功能将保留，并可以改用。
* 由于采用率较低，UGC视频在此版本中被弃用。 设置另一个Dynamic Media Classic帐户可用作UGC视频的替代解决方案。 请联系Adobe技术支持以设置其他帐户。
* 由于采用率较低，此版本中已弃用文本连字。 连字词典定义了如何分割单词并用连字符进行连字，以支持最佳文本流。
* 由于采用率较低，元数据服务器和相似性搜索在2015年均被弃用。 在公布正式弃用目标日期后，受影响的客户将收到Adobe技术支持的通知。
