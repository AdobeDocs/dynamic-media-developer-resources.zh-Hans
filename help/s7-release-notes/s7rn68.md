## Release highlights {#section-524cffc79c1f4bb38627a5ecccd0bf52}

* 增加了响应式Web设计支持——基于断点的响应式图像服务以及基于页面布局动态调整大小的响应式HTML5查看器（图像集、视频、混合媒体、电子目录、旋转集、样式集、缩放）。
* Adobe Analytics视频报告集成。
* 高级HTML5查看器自定义— 使用嵌入网页创建自定义查看器交互。
* 图像增强——添加了将图像裁剪到嵌入路径的投放功能。

## 查看器 {#section-609885a9c8514a4095da74a0b526138d}

更新了以下内容：

* Windows操作系统的最低系统要求现在是Windows 7。
* Internet Explorer的最低系统要求现在为版本9。
* Mac OS的最低系统要求现在为Mac OS 10.8。
* Safari的最低系统要求现在为版本6。

重新分析了以下内容：

* 支持响应式设计的HTML5查看器。
* HTML5查看器使用CSS输入和大小标记控制桌面和移动设备上的显示。

现在支持以下内容：

* 针对HTML5视频查看器和混合媒体查看器的Adobe Analytics视频报告集成。
* 事件跟踪同一页面上的多个HTML5查看器。
* 已添 `SetIndicator` 加到HTML5缩放查看器和HTML5混合媒体查看器中，以可视方式显示移动设备上的色板。
* 添加 `getComponent` 了允许外部访问HTML5查看器组件的API。
* 作为单个JSON对象传递的HTML5查看器配置参数。
* 将WebVTT字幕作为JSON对象传递，无需规则集配置。
* HTML5 eCatalog Viewer中用于将图像映射关联到外部页面的JavaScript模板。
* 链接到HTML5 eCatalog查看器中的页面。
* `stagesize` 到HTML5弹出查看器。
* `initComplete` HTML5查看器的处理程序，以在查看器后触发函数。
* HTML5查看器的分析处理程序，用于支持第三方跟踪。
* 添加了“全选”按钮，可更轻松地获取用于社交共享的链接和嵌入代码。
* 为FlyoutZoomView添加了调整大小功能。
* 在与响应式设计结合使用时，为FlyoutZoomView设置图像重新加载断点。
* 增加了调整音量的单击支持。
* SpinView和ZoomView，在不与查看器交互时将手势应用于嵌入网页。
* 为SpinView `unload()` 和ZoomView添加了API，以允许在运行时处理组件。
* 在不与查看器交互时将手势应用于嵌入网页的视频查看器。
* 为ZoomView设置初始缩放区域。
* ZoomView在无法缩 `IconEffect` 放时隐藏组件

另请参 [阅Scene7查看器参考指南](/help/aem-viewers-ref/t-installing-the-viewers.md)。

**HTML5查看器SDK**

The *Adobe Scene7 HTML5 Viewers SDK* is available as part of the SDK download from Adobe Developer Connection.

[更多信息](http://help.adobe.com/en_US/scene7/using/WSd4272150f67705c11b002eec12fcba4dee6-8000.html).

## 成像 {#section-113f27ea04cd4998a324bba71519c93c}

* 支持渐进式JPEG。
* 更改了新客户端的默认USM锐化设置。
* 为重新处理资产作业添加了USM锐化操作。
* 添加了删除图像预设时的警告。
* 在用于响应式设计时设置图像重新加载断点。

## 平台 {#section-7e2e7bd11b81424e8cd939b6675ca729}

* 在摄取之前，现在使用Clam AntiVirus检查传入的文件是否有病毒。 所有失败的文件将立即隔离。
* 延长作业详细信息弹出窗口以包含每个详细记录的时间。
* 已实施API，用于从资产(deleteAssetMetadata)中删除元数据值。
* 将默认搜索行为更改为包括dc.subject。
* 将密码重置机制“忘记密码”更改为多步进程以避免DOS。
* 将忘记密码的电子邮件从HTML转换为纯文本以防止黑名单。
* 消除了默认角色。
* 除了通过服务器目录上传(ftp)和上传URL将内容上传到Scene7 Publishing System的功能外，您现在还可以使用HTTP POST上传内容。 有关使 [用HTTP POST的详细信息](/help/aem-ips-api/c-overview.md) ，请参阅Image Production System API。

## 错误修复 {#section-456515477c8a4aa69309de43292ce5ed}

**查看器**

* 潜在的Flash Fastzoom和Flyout查看器安全问题。
* 由于无法识别的UserAgent,HTML5查看器未加载。
* HTML5 eCatalog查看器在包含图像映射的页面的Internet Explorer中无法使用鼠标拖放。
* HTML5视频查看器阻止访问取消阻止Flash插件。
* HTML5查看器阻止在Firefox浏览器上打印。
* HTML5缩放查看器在启动或重置缩放时显示轻微的抖动。
* 对HTML5和AS3查看器的HTTPS支持。
* autoResize不应位于HTML弹出查看器复制URL/嵌入代码中。
* HTML5视频查看器复制URL中没有videoserverurl修饰符。
* 该参数 `contenturl` 来自发布服务器HTML5查看器预览URL。

**成像**

* 无法在图像发布设置中将调整后的视图添加为默认图像。
* 将colorManagementOptions标签“转换为SRGB（默认）”更改为“默认颜色管理”。
* 在“文件”>“编辑信息”>“选择多个图像资产”中时，编辑信息不可用。

**平台**

* 在您编辑Adobe Analytics配置时添加了警报消息，然后尝试从设置中转出而不保存。
* 编辑计划的上传作业时，项目选项引用重置为“无”。
* 用户角色不应显示“文件移动”文件夹。
* 无法在“详细信息”视图中看到一些选项卡。
* 安装Flashplayer调试版时出现Web到打印编辑错误。
* “Space After”不起作用；它应使用“sa”，目前它是“margb”。
* 初始化公司设置时，Scene7 Publishing System会在API中发送重复 `<defaultValue>` 选项 `<createMetadataField>`。

* Flash插件和Scene7 Publishing System因大量文件夹没有足够的闪存而崩溃。
* 删除“日期／时间”用户定义字段的值时，Scene7 Publishing System不会向IPS（图像发布系统）发送请求。
* 如果启用了Evideo编码选项，则重新处理PDF或PSD文件时会出现“自适应视频编码”警告消息。
* 启用SSL后，无法导出资产。
* 忘记密码电子邮件中显示的密码不正确。
* 任何Scene7 Publishing System Web管理员请求都将重定向到HTTP。
* 如果未选择任何预设，则在上传期间将应用所有批量集预设。
* 上传s7info内容时，Scene7 Publishing System通过HTTP请求s7info。
* “忘记密码”电子邮件中的URL是HTTP，即使它来自SSL Scene7 Publishing System。
* 特殊字符在HTML格式的“忘记密码”电子邮件中进行编码，导致密码无法正确显示。
* 试用帐户的欢迎电子邮件中缺少Scene7 FTP帐户信息
* 同时选择项目选项和裁剪选项时，重新处理作业失败。
* 当作业选项重置为公司默认值时，上传文件不起作用。
* 标签公司未完全显示在Adobe Analytics设置页面中。
* 无法导入元数据。

## 已知问题 {#section-b2608177928b4174b080d89c938bb0d3}

* Adobe Illustrator CC(Creative Cloud)和Adobe InDesign CC的2014年5月更新预计会引入与创建Scene7 FXG的增效工具的不兼容性。 这些不兼容性会影响Scene7 Web-to-Print解决方案的用户。 如果您使用Scene7 Web-to-Print，请不要将Illustrator和InDesign升级到此Creative Cloud更新。 请继续使用2014年5月之前的版本。 Adobe Systems预计，在2014年秋季，Scene7 FXG创建与较新版本的Illustrator和InDesign的兼容性将得到恢复。
* 当通用查看器预设包含禁用的默认AS2预设时，“嵌入代码”按钮不起作用。
* 重新处理自适应视频集始终会创建新的自适应视频集。
* 在详细信息中添加“时间”后，“作业详细信息”对话框的用户界面在右侧被截断。

## 不再支持功能 {#section-7b04b6fdb8d1473880ccaf84211254ad}

* Scene7 6.9秋季版本中将取消支持文本连字符连接。 连字符词典定义了如何拆分单词并用连字符连接以支持最佳文本排列。
* 自2014年1月31日起，Scene7正式终止对DHTML查看器平台的支持。

   有关此更改的更多信息，请参阅以下常见问题解答网站：

   [http://dev.day.com/content/docs/en/cq/current/administering/integrating_with_omniture/scene7/dhtml-viewer-endoflifefaqs.html](http://dev.day.com/content/docs/en/cq/current/administering/integrating_with_omniture/scene7/dhtml-viewer-endoflifefaqs.html)

* 此版本中已弃用动态(Flash)横幅。 随着其他用于创建、管理和交付个性化网站横幅的技术的出现，此功能的采用率一直较低。 Scene7动态(Flash)横幅功能将继续在服务器端工作，但将不再支持它。
* 提醒：自上一个Scene7版本起，已正式取消支持元数据服务器和关联API的使用。
* Scene7 Creative Suite扩展（Flash面板）将于2014年底弃用。 内容摄取工作流的进一步开发侧重于Adobe Experience Manager Dynamic Media和Creative Cloud。 对于Scene7,Scene7 Publishing System中的上传和导出功能将保留并可用于替代。
* 从2014年5月起，当Scene7中的缓存失效完成时，Akamai不再支持电子邮件通知。 CDN失效的典型处理时间约为8分钟。 如果请求队列较大或URL数量较大，则处理时间可能会稍长一些。
* 类别详细信息视图中URL和嵌入代码选项卡中的“通用预设”。
* RTMP流，用于视频细节视图。
* 来自Scene7 Publishing System的“Email Communications”和SendMailSimple.jsp servlet。
* Media Portal用户欢迎电子邮件模板中的Scene7 FTP帐户部分。
* dc.title，因为它导致IPS（图像发布系统）实例关闭。
* 在HTML5查看器上以全屏模式显示的社交功能。
* 在HTML5查看器上以全屏模式显示的“关闭”按钮。

## Scene7 SDK 2.7.1中的新增功能和错误修复 {#section-b82b4f546ce44a769e94fb1493511035}

**常规**

* 提高了对iOS6的最低要求。
* 修复：在全屏模式下，Safari工具提示会从屏幕上移开。
* 重新构建工具提示以解决各种错误。

**按钮**

* 为默认按钮样式添加了对CSS的支持。

**容器**

* 修复：从全屏返回并交换资产后容器大小不正确。
* 修复：如果通过CSS变换移动容器，则容器坐标不正确。

**电子邮件共享**

* 修复：使用“删除”按钮发生的工具提示对象。
* 修复：滚动条工具提示显示在用户界面后面并被裁剪。
* 修复：“发送电子邮件”按钮后面显示的“取消”工具提示。

**嵌入共享**

* 修复：嵌入代码文本不会绕排，并显示在Internet Explorer上的滚动条上。

**FlyoutZoomView**

* 修复：弹出事件不在Internet Explorer 8上调度。

**可变音量**

* 修复：体积块工具提示显示在体积块条下。

**調色板**

* 修复：当嵌 `selectSwatch` 入到函数中时不触发的方 `initViewer()` 法。

**SocialShare**

* 增加了对bearing修饰符的支持，以控制共享按钮的布局位置。
* 添加了事件以在与社交功能交互时控制暂停／播放。
* 修复：“社交共享”图标不会随着“缩放” `controlbar` 淡出。
* 修复：社交共享图标在初始淡入Galaxy SII时不显示。
* 修复：在iPad3上，社交共享模式对话框不以全屏模式显示。
* 修复：在Safari 5.1的Mac上，社交共享模式对话框不以全屏模式显示。
* 修复：社交共享模式对话框在滚动后显示在屏幕外。

**ThumbnailGridView**

* 修复：ThumbnailGridView不发送 `AssetEvent.ASSET_CHANGED`。

**VideoPlayer**

* 增加了对修饰符 `initialBitRate` 的支持，可选择用于在桌面上初始回放的比特率。
* 实施了iPhone视频代理以解决iPhone设备上的视频问题。
* 增加了对事件监听器的支持，以在调用社交功能时暂停播放。
* 现在，设置为自动的播放模式在Safari桌面上使用本机播放器。
* 修复：HTTPS协议使用Flash播放在IE8上生成安全警告。
* 修复：在使用Flash回放的Safari 5上，视频在全屏模式下无法正确调整大小。
* 修复：在iPhone上播放视频后，无法选择媒体集中的色板。

**ZoomView**

* 修复：在Android上放大图像或旋转设备后显示的伪像。

## S7OnDemand 5.0.1中的新增功能和错误修复 {#section-ba3a595134144711bf27dd8dca249e5d}

* 修复：对HTML5和AS3查看器的HTTPS支持。
* 添加了对HTML5查看器嵌入代码的响应式支持。
* HTML5查看器的嵌入代码将配置参数作为单个JSON对象传递。
