## Scene7 Publishing System 6.9发行说明 {#section-2ed72b7902be4509876d81f25b5632ce}

6.9版本提供了新的高级eCatalog查看器、内联缩放查看器、视频播放器等。 这些新功能允许您设计和提供丰富、响应迅速的交互式体验，从而提高所有数字渠道的转化率。

* **高级eCatalog查看器** — 通过目录、页面动画、图像映射叠加图等，您现在可以提供更吸引人的在线目录查看体验。 您可以突出显示产品和服务，以便客户可以直接链接到可以提高转化率的页面。
* **新的内联缩放查看器** — 允许轻松地在窗口中缩放，而无需单击任何按钮，因此您可以缩放并且仍保持页面上所有产品详细信息的可见性。
* **视频播放器增强功能** — 现成的视频播放器现在包括用于连续视频播放和章节导航的自动循环功能，以便轻松地查找长格式视频中的内容。 混合媒体视频播放器现在支持字幕功能，以便在全球范围内使用。
* **响应式设计**  — 通过为每个断点定义不同的图像预设或修饰符，进一步优化响应式成像体验的质量。
* **查看器自定义**  — 您现在可以动态设置和动态修改查看器样式和行为。
* **Visual Configurator的安全预览**  — 在将个性化体验推送到生产环境之前，安全地预览和测试个性化体验的可视化渲染。
* **单点登录(SSO)功能**  — 现在，您可以从Adobe Experience Cloud无缝启动Scene7，并使用Adobe ID登录到Scene7。

   如果要设置SSO以便在Scene7中使用，请联系Adobe技术支持。 准备好以下信息，以便将Experience Cloud帐户和Scene7帐户关联在一起：您的Adobe Experience Cloud组织名称、您的Scene7公司名称以及您所在的地理区域。

## 视频 {#section-69101b28504543a7ae8a9346d381f913}

* 一款新的开箱即用的“自适应视频”编码预设，现在可与任何纵横比配合使用，以创建视频以交付到移动设备、平板电脑和桌面。 使用此新预设编码的上传源视频将设置为固定高度，而宽度会自动缩放以保留视频的长宽比。 请参阅[用于编码视频文件的视频预设](https://experienceleague.adobe.com/docs/dynamic-media-classic/using/setup/application-setup.html?lang=en#setup)。

* 在使用iOS 7的iPad上，视频的最后一帧可能会失真。

## Scene7 Publishing System 6.9的错误修复 {#section-86dc0e8eb85241dbb449221becb12888}

* 导出的资源总数将比“导出作业”日志详细信息中的原始数量大1。
* 在按批次集预设生成的1d旋转中，当旋转集编辑器打开时，它应该只显示一次警告。
* 无法预览现成的HTML5视频查看器预设。
* 即使重写了规则，Scene7 Publishing System仍会继续记住该组。
* Scene7 Publishing System未正确显示某些正则表达式代码以重复执行。
* 如果未选择任何批次集预设，则在上传时应用所有批次集预设。

## 不再支持的功能 {#section-b061ad3d170a45ee8e694f997fd5d8ef}

* 由于Scene7应用程序不再支持Creative Suite面板，因此本版本已弃用Creative CloudFlash扩展。 内容摄取工作流的进一步开发侧重于Adobe Experience Manager Dynamic Media和Creative Cloud。 对于Scene7，Scene7 Publishing System中的上传和导出功能仍可保留并改用。
* 由于采用率较低，此版本已弃用UGC视频。 设置另一个SPS帐户可以作为UGC视频的替代解决方案。 联系Adobe技术支持以设置其他帐户。
* 由于采用率较低，本版本已弃用文本断字。 连字词典定义了如何拆分单词和用连字来支持最佳文本流。
* 由于采用率较低，元数据服务器和相似性搜索在2015年被弃用。 当正式弃用目标日期宣布时，Adobe技术支持会通知受影响的客户。
* 提醒：自2014年1月31日起，Scene7正式终止对DHTML查看器平台的支持。
